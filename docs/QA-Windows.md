# Ручная проверка на Windows

Быстрый чек-лист, чтобы убедиться, что платформенные части работают.
Сначала запустите `python -m app.diagnose` и приложите вывод.

## Обнаружение
- [ ] «Добавить приложение» показывает установленные программы (Chrome, VS Code…).
- [ ] Системные утилиты НЕ показываются (Character Map, Recovery Drive, dfrgui…).
- [ ] Игры Steam видны в списке; запускаются по клику (открывается Steam).
- [ ] Игры Epic видны (если установлен Epic Games Launcher).
- [ ] У добавленных приложений/игр отображается их настоящая иконка/обложка.

## Запуск и статус
- [ ] Клик по плитке запускает приложение.
- [ ] Индикатор «Запущено» загорается у запущенных программ (в т.ч. запущенных вне Centurio).

## Горячие клавиши
- [ ] Ctrl+1…9 запускают закреплённые приложения, даже когда окно свёрнуто.
- [ ] Своя горячая клавиша из «Изменить» срабатывает глобально.

## Трей / окно / автозапуск
- [ ] Кнопка «свернуть в трей» прячет окно; клик по значку в трее возвращает.
- [ ] «Выход» в меню трея закрывает приложение.
- [ ] «Автозапуск с Windows» добавляет/убирает запись (реестр `…\Run`).

## Меню и удаление
- [ ] Правый клик по плитке открывает меню (Открыть / Избранное / Быстрый запуск / Показать в папке / Изменить / Удалить).
- [ ] Удаление приложения и категории спрашивает подтверждение.

Если что-то не работает — приложите вывод `python -m app.diagnose` и опишите шаг.
